# Marketing Assets

Brand identity, guidelines, and marketing materials.

---

## Structure

| Folder | Purpose |
|--------|---------|
| `brand/` | Core brand identity assets |
| `brand/colors/` | Color palettes and definitions |
| `brand/guidelines/` | Brand style guide documentation |
| `brand/xcore-ci/` | Logo variants and brand mark |
| `elevator_pitch_b5/` | Current pitch materials |

## Brand Assets (xcore-ci/)

| Subfolder | Contents |
|-----------|----------|

| `monochrome/` | Black and white versions |
| `color-variants/` | Logo in different brand colors |
| `transparent/` | Transparent background versions |

| `lockup/` | Logo + wordmark combinations |

## File Formats

| Format | Use Case |
|--------|----------|
| `.svg` | Web, scalable applications, source of truth |
| `.png` | When SVG not supported (export from SVG) |
| `.ico` | Favicon only |

**SVG is the source format.** Generate other formats as needed.

---

## What Belongs Here

✅ Logo files and variants
✅ Brand guidelines documentation
✅ Color palettes
✅ Pitch decks and materials
✅ Marketing copy and messaging

---

## What Does NOT Belong Here

❌ Customer-specific materials
❌ Confidential pricing
❌ Draft social media posts
❌ Temporary campaign assets

---

## Adding New Assets

1. Follow naming convention: `xcore-ci-[variant]-[color].svg`
2. Update guidelines if introducing new variant
3. Place iterations in `_archive/iterations/` if exploring options

# Xonaix Brand Guidelines

**Version:** 1.0.0  
**Last Updated:** 2025-12-23

---

## Overview

This document defines the visual identity standards for Xonaix. All public-facing materials must conform to these guidelines.

## Logo

### Primary Logo

- File: `logos/xonaix-primary.svg`
- Use on light backgrounds
- Minimum size: 120px width

### Logo Variations

- `logos/xonaix-dark.svg` - For dark backgrounds
- `logos/xonaix-icon.svg` - Icon only, for favicons and small spaces

### Clear Space

Maintain clear space around the logo equal to the height of the "X" in Xonaix.

### Don'ts

- Don't distort or rotate the logo
- Don't change logo colors outside approved palette
- Don't add effects (shadows, gradients, outlines)
- Don't place on busy backgrounds

## Typography

### Primary Font

**Inter** - Used for all UI text

- Headings: Inter Bold (700)
- Body: Inter Regular (400)
- Emphasis: Inter Medium (500)

### Monospace Font

**JetBrains Mono** - Used for code

### Scale

| Element | Size | Weight | Line Height |
|---------|------|--------|-------------|
| H1 | 48px | 700 | 1.2 |
| H2 | 36px | 700 | 1.25 |
| H3 | 24px | 600 | 1.3 |
| H4 | 20px | 600 | 1.4 |
| Body | 16px | 400 | 1.5 |
| Small | 14px | 400 | 1.5 |
| Code | 14px | 400 | 1.6 |

## Voice & Tone

### Brand Voice

- **Confident** - We know our technology
- **Clear** - No jargon without explanation
- **Trustworthy** - Honest about capabilities and limitations
- **Forward-thinking** - Focused on the future

### Tagline

*"Intelligence, evolved."*

### Writing Guidelines

- Use active voice
- Keep sentences concise
- Explain technical concepts simply
- Be helpful, not condescending

## Applications

### Website

See `xonaix-web` for current implementation.

### Documentation

- Use brand colors for syntax highlighting
- Include logo in headers
- Follow typography scale

### Marketing Materials

- All materials require brand review
- Use approved templates
- Maintain clear space around logo
